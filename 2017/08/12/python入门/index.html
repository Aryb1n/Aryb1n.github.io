dict

py里可映射类型都是dict实现的

dict是{key: value}形式, 要求key是可散列数据结构

可散列数据结构

  • 在对象生命周期里,他的散列值不变
  • 对象要实现hash()方法
  • 要有qe()方法

python里的不可变类型是可散列数据结构, 对于tuple, 要tuple包含的所有元素都是不可变类型的时候他才算是可散列的
